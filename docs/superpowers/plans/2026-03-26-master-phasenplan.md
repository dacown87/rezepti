## Vision

### 10x Check
Die 10x-Version von Rezepti ist ein vollständiger KI-Kochassistent: Rezept fotografieren oder URL einfügen → sofort strukturiert und einsatzbereit → am Herd Schritt-für-Schritt kochen → Einkaufsliste für die Woche → Wochenplan mit Kalorienübersicht. Alles lokal, keine Cloud, kein Abo.

### Dream State (12 Monate)
```
CURRENT STATE                    DIESE PHASEN               12-MONATS-IDEAL
─────────────────────────────    ──────────────────────     ──────────────────────────────
• Import von URLs ✅              Phase 1: Polished          • Vollständiger Kochassistent
• Rezeptanzeige ✅                  Core                     • Foto → sofort strukturiert
• Serving-Skalierung 80% ✅      Phase 2: Cook + Mobile     • Einkaufsliste + Wochenplan
• Keine Kochführung ❌           Phase 3: Daily Use         • Rating + Notizen
• Keine Einkaufsliste ❌         Phase 4: Share + Smart     • PDF + QR-Karte
• Kein Foto-Import ❌            Phase 5: Growth            • Wochenplaner + Scan
• Kein Login ❌                                             • Login optional
```

## NOT in scope

- Kommentare / Social Feed — ohne Multi-User kein Mehrwert
- KI-Rezeptgenerierung (nur Suche in Sammlung, Phase 4)
- Pinterest / Facebook Import — niedrige Priorität, Extraktion-Pipeline generisch
- Rating-System mit Community — Personal-Tool, nur lokal

## What already exists (Leverage)

- `src/processors/llm.ts` — Groq Vision ist bereits konfigurierbar → Foto-Import baut direkt drauf
